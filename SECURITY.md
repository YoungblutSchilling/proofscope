# Security Policy

## Supported version

ProofScope is maintained as a single deployed service. The current production version is supported.

## Reporting a vulnerability

Please do not open a public issue for a suspected security vulnerability. Contact the repository owner privately with a clear reproduction, impact assessment, and affected endpoint.

We will acknowledge a report within 72 hours, investigate it, and coordinate a fix before public disclosure.

## Data handling

ProofScope reads public GitHub metadata only. It does not accept repository write credentials, execute repository code, or retain user-supplied repository contents.
