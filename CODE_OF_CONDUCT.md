# Code of Conduct

## Our pledge

We welcome constructive participation from people of all backgrounds. Discussion should remain factual, respectful, and focused on making evidence-first developer tools better.

## Expected behavior

Use inclusive language, give actionable feedback, respect privacy, and disclose conflicts of interest when reviewing repository assessments.

## Unacceptable behavior

Harassment, personal attacks, discriminatory language, sharing private credentials or repository contents, and attempts to manipulate evidence are not acceptable.

## Reporting

Report conduct concerns privately to the repository owner. Reports will be handled respectfully and with appropriate confidentiality.
